# Machine Learning & Mathematical Foundations

This repository contains notes, assignments, and code implementations for a comprehensive course on Machine Learning. The coursework covers the mathematical foundations (Calculus, Linear Algebra, Probability) and progresses through supervised and unsupervised learning algorithms.

## Topics Covered

### Part I: Mathematical Foundations & Optimization
* **Lecture 1:** Review of Functions, Derivatives, and Integrals
* **Lecture 2:** Basic Vector/Matrix Notations and Operations
* **Lecture 3:** Norms, Matrix and Vector Calculus (Part 1)
* **Lecture 4:** Matrix Calculus (Part 2)
* **Lecture 5:** Gradient Descent

### Part II: Probability, Statistics & Preprocessing
* **Lecture 6:** Data Preprocessing and Multivariate Regression
* **Lecture 7:** Review of Probability Notations and Concepts
* **Lecture 8:** Review of Statistical Notation and Concepts
* **Lecture 9:** Feature Importance and Regularization
* **Lecture 10:** Probability Redux: Recognizing Distributions
* **Lecture 11:** Maximum Likelihood (MLE) and Kernel Density Estimation (KDE)
* **Lecture 12:** Sampling Methods
* **Lecture 13:** Basic Information Theory

### Part III: Classification Algorithms (Supervised)
* **Lecture 14:** KNN and Logistic Regression
* **Lecture 15:** Support Vector Machine (SVM)
* **Lecture 16:** Decision Trees and Random Forests

### Part IV: Dimensionality Reduction & Clustering
* **Lecture 17:** Covariance and Correlation
* **Lecture 18:** Principal Component Analysis (Unsupervised)
* **Lecture 19:** Linear Discriminant Analysis (Supervised)
* **Lecture 20:** Clustering (K-Means, Agglomerative, PCA)

**Libraries:**
* **Data Manipulation:** NumPy, Pandas
* **Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-Learn
* **Statistics & Math:** SciPy, Statsmodels, Autograd
